### Usage
``` jsx live
<Gradient color={Colors.green30} numberOfSteps={4}/>
```
## API
### color
The color of the gradient
`string ` 

### numberOfSteps
The number of steps
`number ` 

### style
Additional style to the component
`ViewStyle ` 

### type
hue | lightness | saturation
`GradientTypes `
