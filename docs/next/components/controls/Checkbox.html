CheckboxCheckboxes allow the user to select one or more items from a set. They can be used to turn an option on or off, or to indicate approval or verification.

Spec
SizeThe checkbox comes only in one size, and cannot be resized. Size: 24x24
PositionFor screens requiring multiple selections, use the Multi-Select List. If a Checkbox is used to indicate approval or verification, place it with a label to its right.
Checkbox GroupUse ListItem component to group multiple options for the user to choose from.
