ChipsInput

Chips input component
(code example)

info

This component extends TextField props.

tip

This component support typography modifiers.

Usage#

<ChipsInput placeholder={'Placeholder'} chips={[{label: 'Falcon 9'}, {label: 'Enterprise'}, {label: 'Challenger', borderRadius: 0}]}/>

API#

chips#

List of tags. can be string boolean or custom object when implementing getLabel
ChipsInputChipProps[]

containerStyle#

Component's container style
ViewStyle

defaultChipProps#

Style your chips
ChipsInputChipProps

disableTagAdding#

If true, tags adding UX (i.e. by 'submitting' the input text) won't be available
boolean

disableTagRemoval#

If true, tags removal UX won't be available
boolean

getLabel#

Callback for extracting the label out of the tag item
(tag: ChipType) => any

hideUnderline#

Should hide input underline
boolean

inputStyle#

Custom styling for the text input
RNTextInputProps['style']

leftElement#

Custom element before the chips, for example 'search' icon, 'To:' label etc'
JSX.Element | JSX.Element[]

maxHeight#

Chips inside a ScrollView
number

maxLength#

Maximum numbers of chips
number

onChangeTags#

Callback for 'onChangeTags' event
() => void

onCreateTag#

Use chips instead. callback for creating new tag out of input value (good for composing tag object)
(value: any) => void

onTagPress#

use chips instead. callback for when pressing a tag in the following format (tagIndex, markedTagIndex) => {...}
(index: number, toRemove?: number) => void

renderTag#

use chips instead. Callback for custom rendering tag item
(tag: ChipType, index: number, shouldMarkTag: boolean, label: string) => React.ReactElement

scrollViewProps#

Chips with 'maxHeigh' is inside a ScrollView
ScrollViewProps

selectionColor#

The color for the selection state
string | number

tagStyle#

Custom styling for the tag item
ViewStyle

tags#

use chips instead. List of tags. can be string boolean or custom object when implementing getLabel
ChipType[]

validationErrorMessage#

Validation message error appears when tag isn't validate
string

value#

The input's value
any
